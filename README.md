# Resort Booking Application (Full Stack)

A modern **Resort Booking Application** built using **Spring Boot + MySQL (Backend)** and **React + Tailwind CSS (Frontend)**.

This project demonstrates real-world full stack development including REST API design, database relationships, and frontend-backend integration.

---

##  Features

###  User Features
- Browse available resort rooms
- View room details
- Book rooms
- Responsive modern UI
- Smooth loading animations

###  Backend Features
- RESTful API development
- Room management
- Booking management
- JPA entity relationships
- MySQL database integration

---

##  Tech Stack

###  Frontend
- React (Vite)
- Tailwind CSS
- Axios
- Component-based architecture

###  Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
